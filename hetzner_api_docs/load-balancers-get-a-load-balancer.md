### Get a Load Balancer

Gets a specific Load Balancer object.

#### HTTP Request

`GET /load_balancers/{id}`

#### Path Parameters

- idinteger (int64) required

  ID of the Load Balancer.

#### Response

- load\_balancerobject  required
