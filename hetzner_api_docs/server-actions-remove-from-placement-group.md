### Remove from Placement Group

Removes a Server from a Placement Group.

#### HTTP Request

`POST /servers/{id}/actions/remove_from_placement_group`

#### Path Parameters

- idinteger (int64) required

  ID of the Server.

#### Response

- actionobject  required
